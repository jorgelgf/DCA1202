Detailed Description

The Plotter class is responsible for the construction and modification of a Sculptor, using a graphical interface. It contains the graphical interface and numerous funcitons that create a 3D sculpture.

void Plotter::drawShape ( int  shape,
bool  mousePressed 
)

drawShape is where the mouse events relate to a Sculptor: after retrieving the X,Y,Z positions on mouse events, they draw a geometric figure given a chosen figure, state of mouse clicked and slider's values.

Parameters
shapesignifies which shape will be drawn: PutVoxel, CutVoxel, PutBox, CutBox, PutSphere, CutSphere, PutEllipsoid, CutEllipsoid.
mousePressedis a bool that activates the draw action.
